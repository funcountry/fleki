# Receipts

This checked-in directory mirrors command receipts as reference content.

The live receipts live under `<resolved_data_root>/receipts/`.

Some checked-in receipt exemplars may predate the current exact-only contract. Treat them as historical examples, not live command truth.

This directory holds append-only command receipts for:

- `save`
- `search`
- `trace`
- `rebuild`
- `status`
